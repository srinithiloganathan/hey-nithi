# 🧠 AI Desktop Assistant (Offline)

A lightweight, offline personal assistant built using Python that can understand natural language commands and perform real system-level actions like scheduling reminders, opening applications, and launching web services.

---

## 🚀 Features

### 🗣️ Natural Language Commands
- Understands commands like:
  - `schedule meeting tomorrow at 10am`
  - `meeting in 1 min`
  - `open chrome`
  - `open gmail`

---

### 📅 Smart Scheduling (Offline)
- Parses human-friendly time using NLP
- Stores events locally (`events.json`)
- Triggers:
  - 🔔 Windows notifications  
  - 🔊 Voice alerts  

---

### 💻 System Automation
- Open desktop applications:
  - VS Code
  - Postman
  - MongoDB (Compass)
  - OpenVPN
  - Chrome

---

### 🌐 Browser Automation
- Open:
  - Gmail
  - WhatsApp Web
  - Any custom link

---

### 🔊 Voice Feedback
- Uses offline TTS (`pyttsx3`)
- Gives real-time spoken responses
